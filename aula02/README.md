### 👉 Vector

Estrutura de dados que funciona como lista de objetos
com *crescimento dinâmico*. Trabalha como um vetor, introduzindo
novos elementos ao final da lista e podemos manipular os dados
através de *índices*.

### 👉 ArrayList

Um array dinâmico, mesmas características do Vector, mas sua estrutura é mais
**performática**.

### 👉 LinkedList

Lista que funciona de forma *encadeada*. Sua performance é melhor em métodos
de alteração de registros.  

### 👉 HashMap

Uma estrutura de dados em que armazena os dados associados
a uma chave única. É composto por uma **chave** e um **valor**. Em grande volumes 
de dados, é a estrutura mais recomendada a ser usada.
