Para essa atividade escolhi usar **Queue**. 🏃🏃🏃

Para adicionar e remover tarefas, a **Queue** é ótima, pois funciona
com o princípio *"First-In-First-Out"*, então as tarefas serão processadas
na ordem que foram adicionadas, e na conclusão de uma tarefa, basta apenas
remover o elemento do início da fila.

Outro motivo pelo uso da **Queue** foi o método *.peek()*, pois ele retorna
o próximo elemento da fila sem precisar percorre-la toda.

A **Queue** também é uma ótima escolha para garantir que o sistema
seja escalável, pois consegue lidar com um grande número de tarefas.
